1. python manage.py runserver - запустить сервер
2. python manage.py makemigrations - если была добавлена новая модель, то её нужно занести в миграции
3. python manage.py migrate - исполнить все миграции

Модели

Все модели описываются в models.py, а потом регистрируются в админке (admin.py) с дополнительным описанием

Админская панель - /admin

Чтобы добавить страничку сайта - нужно сделать View для неё (view.py)
Потом зарегистрировать путь в urls.py

все пути для конкретного приложение регаются в локальном urls.py, а потом заносятся одной строчкой в глобальный
